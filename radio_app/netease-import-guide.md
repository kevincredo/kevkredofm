# KevinCredo FM 网易云导入指南

这个功能用于把 FM 里的红心歌曲反向创建成网易云音乐歌单。网页可以放在 Netlify 上，但网易云扫码登录和创建歌单仍通过用户自己电脑上的本机助手完成。

## 本机准备

1. 安装 Node.js 18 或更新版本。
2. 在项目根目录运行本机网易云助手：

```bash
npm run netease:api
```

3. 保持这个窗口打开。默认助手地址是：

```text
http://127.0.0.1:3000
```

如果用户只是打开 Netlify 网站、没有下载整个项目，也可以先下载网站自带的助手脚本：

```bash
curl -fsSLO https://kevkredofm.netlify.app/netease-local-helper.mjs
node netease-local-helper.mjs
```

## 网页里操作

1. 打开 KevinCredo FM。
2. 打开 `NETEASE OUT`。
3. 点击 `扫码登录`，用网易云音乐 App 扫码并在手机上确认。
4. 点击 `生成草稿`，检查要导入的红心歌曲数量和歌单名称。
5. 点击 `确认上传`，在确认弹窗里确认后，网页会通过本机助手创建网易云歌单并批量添加歌曲。

`高级设置与备份` 里可以改歌单名称、检查本机助手、复制歌曲 ID、下载草稿 JSON。

## 上传到 Netlify 后怎么用

1. 把项目上传到 Netlify，发布目录保持 `radio_app`。
2. 用户打开 Netlify 页面。
3. 用户在自己的电脑上运行 `npm run netease:api`，或者从网站下载 `netease-local-helper.mjs` 后运行。
4. 用户在网页 `NETEASE OUT` 里扫码登录并导入。

这个设计的重点是：Netlify 只负责展示网页和保存 FM 红心数据；网易云登录态和写歌单请求留在用户自己的电脑上。

如果扫码显示 `Failed to fetch`，通常是网页没有连上当前设备的本机助手。请确认终端窗口还开着，并且网页里的本机助手地址是 `http://127.0.0.1:3000`。手机或另一台电脑打开 Netlify 页面时，`127.0.0.1` 指向的是那台设备自己，不能连接你电脑上的助手。

## 安全边界

- 真实导入只允许 `localhost`、`127.0.0.1` 或 `::1` 这类本机地址。
- 网易云登录态只用于本机助手，不会保存到 Netlify。
- 创建歌单前会弹出确认框；取消后不会修改网易云账号。
- 如果网易云接口返回歌曲无版权、下架、地区不可用，可能会出现部分歌曲无法添加。
